# kernelcache_decryptor
Kernel Cache Decryption for iOS 12 or laters working on mac M1

# Requirements:
* **macOS**:
```bash
brew install lzfse, libtasn1
```
# Run:
```
./kernelcache_decryptor <kernelcache>
```
